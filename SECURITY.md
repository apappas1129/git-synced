# Security Policy

## Supported Versions

Only the latest published version on npm is supported.

## Reporting a Vulnerability

This tool generates and executes shell scripts (`script.sh`) on your machine using data
you provide (a saved GitHub profile page, committer name/email/message). If you find a
security issue — e.g. a way for untrusted input to escape the generated commands —
please open a [GitHub issue](https://github.com/apappas1129/git-synced/issues) or
contact the maintainer directly at apappas1129@gmail.com rather than disclosing it
publicly first.
